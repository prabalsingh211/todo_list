# To-Do List Application - Angular Assessment

## Project Overview

This is a To-Do List application built with Angular 19 as part of the frontend developer. The application follows all specified requirements including best practices, modular architecture, and comprehensive testing.

## Features

- Create, Read, Update, Delete (CRUD) tasks
- Responsive design for desktop and mobile
- Search and filter functionality
- Pagination

## Tech Stack

- **Framework**: Angular 19
- **Package Manager**: npm
